# 🚀 LangChain Learning Repository

A collection of hands-on projects and examples demonstrating the core concepts of **LangChain**, including Large Language Models (LLMs), Chat Models, Embedding Models, and document similarity search.

This repository is intended for learning and experimenting with LangChain components using Python and OpenAI models.

---

## 📂 Repository Structure

```
LangChain/
│
├── 1.LLMs/
│   └── Examples using Large Language Models
│
├── 2.ChatModels/
│   └── Chat model implementations
│
├── 3.EmbeddingModels/
│   └── Embedding model examples
│
├── document_similarity/
│   └── Semantic document similarity project
│
├── requirements.txt
├── test.py
└── README.md
```

## 📦 Installation

### Clone the repository

```bash
git clone https://github.com/SudarshanAhire/LangChain.git
```

```bash
cd LangChain
```

### Create Virtual Environment

Windows

```bash
python -m venv venv
venv\Scripts\activate
```

Linux / macOS

```bash
python3 -m venv venv
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

## 🔑 Environment Variables

## 📚 Modules

### 1. LLMs

Learn how to interact with Large Language Models using LangChain.

Topics include:

- Prompting
- Model invocation
- Response generation

---

### 2. Chat Models

Examples of chat-based AI interactions.

Topics include:

- Chat prompts
- Conversations
- AI assistants

---

### 3. Embedding Models

Generate vector embeddings for text.

Topics include:

- Text embeddings
- Vector representations
- Semantic search

---

### Document Similarity

A practical implementation of semantic document similarity using embeddings.

Features:

- Convert documents into vectors
- Compute similarity scores
- Compare semantic meaning instead of exact keywords
